# Python Calculator

A simple calculator built using Python.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Exponentiation (**)
- Float Number Support

## Technologies Used
- Python

## How to Run
1. Download or clone this repository.
2. Run calculator.py (or main.py).
3. Enter your choice and numbers.
